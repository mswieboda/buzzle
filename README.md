## Installation

Requires Crystal to be installed

```
$ brew install crystal
```

Install crystal shard dependencies ([cray](https://github.com/tapgg/cray))

```
$ shards install
```

Run the Makefile to build/compile in Crystal, and run the executable (from `./builds`)

```
$ make
```
